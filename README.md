# College Admin Portal

A simple **College Administration Portal** built using **HTML, CSS, and JavaScript**.
This project is designed to help manage college data such as students, courses, and faculty through an admin dashboard.

---

## 🚀 Features

### 🔐 Admin Login

* Secure login interface for administrators.
* Simple and clean UI.

### 📊 Admin Dashboard

* Sidebar navigation.
* Dashboard layout for managing college resources.

### 👨‍🎓 Student Management

* Add students
* Delete students
* Display students in a table

### 📚 Course Management

* Add new courses
* Delete courses
* Display course list

### 👨‍🏫 Faculty Management

* Add faculty members
* Delete faculty members
* Display faculty list

### 🎨 Modern UI

* Sidebar navigation panel
* Responsive admin dashboard layout
* Styled tables and forms
